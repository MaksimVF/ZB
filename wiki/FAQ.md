# Frequently Asked Questions

## Account Issues
**Q: How do I reset my password?**
A: Use the "Forgot Password" link on the login page.

**Q: Can I change my email?**
A: Contact support to update your email address.

## API Keys
**Q: What permissions should I set?**
A: Use the least privilege principle - only grant necessary permissions.

**Q: How do I rotate API keys?**
A: Use the "Rotate" button on the API Keys page.

## Payments
**Q: What payment methods are supported?**
A: We support YuKassa, Stripe, and PayPal.

**Q: How do I get a refund?**
A: Contact support with your transaction details.
