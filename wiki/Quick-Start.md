# Quick Start Guide

## Registration
1. Go to the [registration page](/register)
2. Enter your email and create a secure password
3. Verify your email address

## Creating API Keys
1. Navigate to the [API Keys page](/api-keys)
2. Click "Create New API Key"
3. Set appropriate permissions
4. Save the key securely

## Making Payments
1. Go to the [Top Up page](/topup)
2. Choose your payment method
3. Select amount or subscription plan
4. Complete the payment process

## Security Setup
1. Visit the [Security page](/security)
2. Enable Two-Factor Authentication
3. Review active sessions
